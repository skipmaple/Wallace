# 合并订单数据到电子器件分类汇总

## 触发条件

用户提供一个淘宝订单导出的 xlsx 文件（如 `订单数据 (5).xlsx`），要求合并/汇总到 `doc/电子器件分类汇总.md`。

## 输入

- 一个 xlsx 文件，位于 `doc/` 目录下，包含淘宝订单数据
- 关键列：商品名称、型号款式、数量、商品链接（列名可能有变体）

## 输出

更新 `doc/电子器件分类汇总.md`，将新订单中的器件按分类追加到对应表格中。

## 执行方式

运行脚本（需要 pandas）：

```bash
python .claude/skills/scripts/merge_order.py "doc/订单数据 (5).xlsx"
```

脚本会自动：
1. 解析现有 `doc/电子器件分类汇总.md` 中的数据和来源文件列表
2. 读取 xlsx 并自动识别列名（商品名称/名称/标题、型号/款式/规格、数量/件数、链接/宝贝链接/URL）
3. 按商品名称关键词分类到 10 个类别（传感器/摄像头、开关/按键、执行机构/机构件、显示/人机界面、电机驱动、电源/供电/配线、紧固件/结构件、连接器/线材、通信/无线模块、其他）
4. 按「商品名称+型号款式」去重，跳过已有记录
5. 追加来源文件名，更新总数量，重新生成 markdown
